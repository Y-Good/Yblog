# Yblog
本项目使用pnpm管理依赖，如果没有安装pnpm，请先安装pnpm
## 安装
```shell
$ pnpm install
```
## 运行
```shell
$ pnpm run blog
```
## 运行端口
- Server运行端口：http://localhost:3001
- Backend运行端口：http://localhost:9999
- Web运行端口：http://localhost:8888

## 其他
- 根目录添加依赖
```
pnpm add <依赖> -w
```
- 子包添加依赖
```
pnpm add <依赖> --filter <子包>
 ```
